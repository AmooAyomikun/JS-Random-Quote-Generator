**Inspirational Quote App**

The Inspirational Quote App is a web-based project built with vanilla JavaScript, HTML, and CSS. It demonstrates DOM manipulation, event handling, browser storage, and interactive UI design. Users can generate random quotes, save their favorites, remove them, copy quotes, share via native browser functionality, or tweet directly.

**Features**

**Random Quote Generator:** Click "New Quote" to display a fresh inspirational quote.

**Save & Remove Quotes:** Users can save favorite quotes and remove them individually.

**Copy Quotes:** Quickly copy quotes to the clipboard.

**Share Quotes:** Use native browser sharing to share quotes across platforms.

**Tweet Quotes:** Post quotes directly to Twitter with a single click.

**Persistent Storage:** Saved quotes remain even after refreshing the page via localStorage.

**Interactive UI:** Buttons and saved quotes dynamically update using JavaScript.
